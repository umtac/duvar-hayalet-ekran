# DUVAR & HAYALET EKRAN

Bu proje, kullanıcıların şifreli ve kendini imha eden görüşmeler yapabildiği özgün bir sosyal medya deneyimi sunar.
- React Native (Expo) ile geliştirilmiştir.
- `HayaletEkran` bileşeni mesajları şifreler, çözer, siler.
- `utils/crypto.js` AES şifreleme altyapısını içerir.

Kurulum:
1. ZIP dosyasını çıkarın.
2. `npm install` komutuyla bağımlılıkları yükleyin.
3. `npx expo start` ile uygulamayı başlatın.

Bu kodları kullanmak serbesttir ancak lütfen özgün fikrin sahibi olarak Ümüt Açığ’ı referans gösterin.
